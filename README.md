# Robô Chef

Aplicação com Frontend (React + Vite + Tailwind CSS) e Backend (Node.js + Express).

## Como executar localmente

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Build para produção (frontend)
```bash
cd frontend
npm run build
```

## Deploy com Vercel
```bash
vercel
```
